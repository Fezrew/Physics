Detailed Description

A scene where physics objects interact with each other. You may add and remove physics objects at will. The scene keeps track of all the objects and checks for collisions, determining their shape in order to do so. This class is in charge of calling all the functions inside the various other classes, and can be cleared of all objects whenever.
